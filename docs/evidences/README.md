## 2. Fluxo de Branches

O projeto utiliza o modelo **Git Flow** com as branches:

- `main` — Código de produção
- `develop` — Branch de integração
- `feature/*` — Branches de funcionalidades

> Para mais detalhes, consulte: [Desenvolvimento vs Produção](../workflow/development-vs-production.md)
