1. Funcionalidades Essenciais
Sistema de Autenticação: Cadastro e login de usuários para personalização da experiência.

Interface de Busca: Campo de pesquisa global para localização de componentes por nome ou modelo.

Categorização de Hardware: Filtros rápidos para Processadores, Placas de Vídeo, Memória RAM, SSDs e Placas-Mãe.

Comparador de Preços em Tempo Real: Listagem de um mesmo produto em diferentes lojas (Kabum, Pichau, Terabyte) com destaque para o menor valor.

Filtros Avançados:

Filtragem por Marca (Ex: Intel, AMD, NVIDIA).

Filtragem por Faixa de Preço (Slider dinâmico).

Filtragem por Capacidade (Ex: 8GB, 16GB, 1TB).

2. Funcionalidades Intermediárias
Cálculo de Economia: Indicador visual de quanto o usuário está economizando em relação ao preço médio ou à segunda loja mais barata.

Dashboard de Ofertas: Seção dedicada a componentes que tiveram queda brusca de preço nas últimas 24 horas.

Integração de APIs/Web Scraping: Desenvolvimento dos scripts para coleta automática de dados das lojas parceiras.

Responsividade Mobile: Garantir que a interface (conforme protótipo) funcione perfeitamente em dispositivos móveis via Flutter.

3. Funcionalidades Desejáveis
Histórico de Preços: Gráfico detalhando a variação do valor do produto nos últimos 30, 60 e 90 dias.

Alertas de Preço: Sistema de notificação via e-mail ou push quando um produto atingir o "Preço Alvo" definido pelo usuário.

Montador de Setup: Ferramenta que verifica a compatibilidade entre as peças selecionadas (Ex: Soquete da placa-mãe vs Processador).

Sistema de Avaliações: Espaço para usuários darem notas e comentários sobre a performance de componentes específicos.
