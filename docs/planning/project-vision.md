# 🎯 Visão do Projeto — PoupePC

---

## 1. Objetivo Principal

O **PoupePC** tem como objetivo principal **simplificar a montagem de computadores gamer**, fornecendo uma plataforma centralizada que automatiza a comparação de preços de componentes de hardware. O sistema busca garantir que o usuário finalize sua configuração com o **maior nível de economia possível**.

---

## 2. O Problema

Atualmente, o mercado de hardware no Brasil é fragmentado entre diversas lojas (Kabum, Terabyte, Pichau, etc.). O processo de comparar manualmente o preço de cada item (Processador, GPU, RAM) em cada site é:

- **⏰ Demorado** — Exige que o usuário abra dezenas de abas no navegador.
- **📉 Ineficiente** — Preços e estoques variam em questão de minutos.
- **🧩 Complexo** — Difícil de visualizar onde o conjunto total da máquina sai mais barato.

---

## 3. Público-Alvo

| Perfil | Necessidade |
|--------|------------|
| **🎮 Gamers e Entusiastas** | Buscam o melhor desempenho pelo menor custo |
| **💼 Profissionais de TI** | Precisam montar estações de trabalho (workstations) |
| **👤 Usuários Leigos** | Precisam de auxílio para entender quais peças oferecem o melhor custo-benefício no momento |

---

## 4. Benefícios e Diferenciais

| Diferencial | Descrição |
|------------|-----------|
| **💰 Economia Real** | Identificação visual de descontos e comparativo imediato entre lojas |
| **🔍 Centralização** | Filtros por marca (Intel, AMD), capacidade e avaliação em uma única interface |
| **⚡ Agilidade** | Redução do tempo de pesquisa para a montagem de um setup completo |
| **📊 Monitoramento** | Atualização diária de mais de 5.000 peças |

---

## 5. Visão de Produto

> *"Ser a principal referência brasileira em comparação de preços de hardware, proporcionando economia real e agilidade para quem monta seu próprio computador."*
